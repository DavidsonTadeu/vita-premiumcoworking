# Vita Premium Coworking - Landing Page

Uma landing page ultra-moderna, altamente interativa e de estética premium para o **Vita Premium Coworking**, localizado na Savassi, Belo Horizonte. 

O projeto foi projetado com foco em alta conversão e experiência imersiva de alto padrão para atração de clientes.

## 🚀 Tecnologias Utilizadas

- **HTML5 & CSS3**: Estrutura semântica com layout responsivo (Grid/Flexbox) e estilização refinada.
- **JavaScript (ES6+)**: Lógica interativa customizada.
- **GSAP (GreenSock Animation Platform)**: Animações de scroll avançadas, efeitos magnéticos em botões e revelações suaves.
- **Three.js**: Fundo interativo tridimensional com partículas dinâmicas no Hero.
- **Lenis Smooth Scroll**: Rolagem ultra-suave.
- **Fontes**: *Clash Display* para títulos imponentes e *Inter* para excelente legibilidade de corpo.

## ✨ Recursos Principais

1. **Design High-End**: Fundo escuro com acentos dourados e cobre, utilizando efeitos de Glassmorphism.
2. **Fundo Tridimensional Interativo**: Efeito de partículas conectadas que reagem à movimentação do mouse (Three.js).
3. **Galeria Dinâmica**: Exibição elegante com imagens reais dos espaços (Salas Privativas, Mesas Compartilhadas, Salas de Reunião, Auditório, etc.).
4. **Tabela de Preços Competitiva**: Três planos estruturados com base no mercado real da Savassi (Essencial, Profissional, Corporativo).
5. **Banner de Cookies & Políticas**: Em conformidade com as boas práticas de privacidade.
6. **Formulário de Contato Inteligente**: Validação robusta e simulação de envio com feedback instantâneo.
7. **Botão de WhatsApp Flutuante**: Acesso direto para o atendimento.
